# Sources & further reading

This is a curated list of the most useful primary references for DSFTool and DSF authoring.

## Primary DSFTool docs / release notes

- **XPTools bundle download page (binaries)**  
  https://developer.x-plane.com/tools/xptools/

- **XPTools 24‑5 Windows zip (includes `README.dsf2text`)**  
  https://files.x-plane.com/public/xptools/xptools_win_24-5.zip

  This archive contains DSFTool instructions, DSF2Text format reference, and a version history through DSFTool 2.4 a1.

## Official DSF format and behavior docs (Laminar)

- **DSF Usage In X‑Plane** (how the sim interprets DSF content, properties, rasters, and validation rules)  
  https://developer.x-plane.com/article/dsf-usage-in-x-plane/

- **DSF File Format Specification** (byte-level DSF spec and high-level architecture)  
  https://developer.x-plane.com/article/dsf-file-format-specification/

- **Scenery Development Documentation hub** (links to multiple file format specs and manuals)  
  https://developer.x-plane.com/docs/scenery/

## Developer blog posts about DSFTool

- **New DSFTool Available** (2012; highlights raster layer support for X‑Plane 10)  
  https://developer.x-plane.com/2012/02/new-dsftool-available/

- **Command-Line Tools Beta Builds** (2020; 64-bit tools, 7z support, 7-plane vertices, set_AGL support)  
  https://developer.x-plane.com/2020/09/command-line-tools-beta-builds/

- **DSF2Text – Just a Link in the Chain** (2005; intent/purpose of DSF2Text)  
  https://developer.x-plane.com/2005/12/dsf2text-just-a-link-in-the-chain/
